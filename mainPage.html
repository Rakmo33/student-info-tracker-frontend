About the Student Information Tracker...      

        Student Information Tracking System is a Web application where User can search for a particular Student of the Institute and get the information along with his/her performance in exams for each semester. All the information is shown in a visually appealing way such as graphs, pie-charts, etc.
        The application also suggests a particular stream / domain in which the student might be interested in based on his/her past performance.
